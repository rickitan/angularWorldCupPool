angularWorldCupPool
===================

- A football pool for the 2014 Brazil Worldcup made with Angular.js

- Una quiniela hecha con Angular.js para el Mundial Brasil 2014

Check it out: 

http://rickitan.github.io/angularWorldCupPool/

To run it locally:

1) bower install
2) grunt serve
3) Go to: http://localhost:9000/
